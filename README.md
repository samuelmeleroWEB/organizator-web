# Organizator.AI ⚡️

👉 **[¡Prueba la Demo Gratis en Vivo Aquí!](https://organizator-web.vercel.app/)**

![Organizator.AI Banner](https://via.placeholder.com/1200x600?text=Organizator.AI+-+Intelligent+Agenda+Generator)

**Organizator.AI** is a smart, AI-powered daily agenda generator designed to optimize your productivity. By taking into account your available work hours, fixed meetings, and pending tasks, Organizator.AI crafts the perfect schedule using advanced AI models like GPT-4o-mini, GPT-4o, and Claude 3.5 Sonnet.

## 🌟 Features

*   **Intelligent Scheduling**: Automatically organizes your tasks applying productivity best practices (e.g., placing high-concentration tasks early in the morning).
*   **Dynamic Time Blocking**: Respects your existing commitments and fixed time blocks (meetings, appointments).
*   **Adaptive Breaks**: Suggests active pauses to maintain your energy throughout the day.
*   **Custom AI Integration**: Support for multiple AI providers. Bring your own API key for OpenAI or Anthropic.
*   **Beautiful UI**: A modern, glassmorphic layout with fluid animations powered by Tailwind CSS and Framer Motion.
*   **Real-time Simulation**: Test the application instantly with the "Demo Gratis" feature.

## 🚀 Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Library**: [React](https://react.dev/) 19
*   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)

## 🛠️ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

*   Node.js (v20+ recommended)
*   npm (or yarn / pnpm)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/samuelmeleroWEB/organizator-web.git
    cd organizator-web
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in your browser**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.

## 🤖 Using the AI

To use the AI generation features in your local environment, click on the **⚙️ Ajustes API** button in the upper right corner of the application:
1. Select your preferred model (OpenAI's GPT-4o-mini, GPT-4o, or Anthropic's Claude 3.5 Sonnet).
2. Enter your personal API key. (Keys are stored locally in your browser's `localStorage` and never sent to our servers).
